# TrustVerify AI

AI-powered document forgery detection and verification system.

## Features
- Upload PDF/Image documents
- Detect fake vs real with confidence score
- AI-generated summary
- Chatbot for document Q&A

## Live Demo
https://trustverify-ai-1.onrender.com

## Tech Stack
- Python (Flask)
- Groq API (Chatbot)
- OCR (Tesseract)
- PDF/Text Processing
